The build process is straightforward: run the build command, get a folder of static files. No complex configuration, no environment variables to manage, no database migrations to run. Just HTML, CSS, and JavaScript files that work anywhere.

Deploy to CloudCannon. Since everything is pre-rendered at build time, you don't need server-side processing. Upload the files, point your domain, and you're live.

The build is fast because there's no runtime to bundle, no database to query, no API calls to make. Everything is already compiled and ready to serve. A typical build takes seconds, not minutes.
